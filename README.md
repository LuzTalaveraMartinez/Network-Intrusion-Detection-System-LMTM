# 🛡️ LMTM: SOC Active Monitoring (Grafana + Loki)

**Desarrollado por:** Luz Maria Talavera Martinez  
**Fecha:** 12 de Abril, 2026

## 📝 Descripción
Este repositorio contiene la infraestructura de **Observabilidad de Seguridad** diseñada para centralizar y visualizar incidentes detectados por el sistema de defensa activa LMTM. Implementa un stack moderno de **Loki y Grafana** optimizado para entornos de recursos controlados.

## 📂 Estructura del Proyecto
* **/configs**: Archivos de configuración YAML para el motor Loki y el agente Promtail.
* **/docs**: Documentación técnica y evidencias de implementación.

## 🛠️ Tecnologías Utilizadas
* **Grafana Loki**: Motor de agregación de logs.
* **Grafana OSS**: Visualización y Dashboards.
* **Promtail**: Log shipper para la ingesta de alertas de Snort.
